# Student Performance Predictor

Predicts student math scores using Linear Regression based on demographic and academic features.

## Dataset
[Students Performance in Exams](https://www.kaggle.com/datasets/spscientist/students-performance-in-exams) — 1000 rows, 8 columns.

## Results
- **MAE:** 4.21
- **R²:** 0.8804

## Features Used
- Gender, Race/Ethnicity, Parental Education, Lunch, Test Preparation
- Reading Score, Writing Score

## Run
```bash
pip install pandas scikit-learn
python main.py
```
